# EMAS 小助手

🔗 **完整文档请查看 → [docs/README.md](docs/README.md)**

面向移动端研发与质量的桌面端辅助工具（Flutter / macOS），通过阿里云 EMAS AppMonitor OpenAPI 拉取崩溃、ANR 等聚合数据，集成 GitLab 堆栈检索、OpenAI 兼容大模型分析等功能。

## 快速开始

```bash
# 1. 安装依赖
flutter pub get

# 2. 运行应用
flutter run -d macos
```
